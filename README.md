# chatdku-agentic-rag

A mini agentic RAG pipeline for document question answering, built for the ChatDKU AI Engineer task.

## Features
- Vector Search
- Keyword Search
- Internet Search
- Bilingual interface (Chinese & English)
- Source-grounded answers with document name and page number

## Tech Stack
- Python
- DSPy
- Local LLM serving with SGLang or vLLM

## Status
This repository is currently under development.
